## API Endpoints
Car Management
  GET	    /api/cars	
  GET	    /api/cars/{car}	->  Show details of a car
  GET	    /api/my-cars	-> Lists cars
  POST	  /api/cars	  ->  Add a new car
  PUT	    /api/cars/{car}	->  Update an existing car
  DELETE	/api/cars/{car}	->  Delete a car
  DELETE	/api/cars/{car}/photo
